# Biblioteca 

> Interfaz gráfica para una biblioteca.

---
> Durante el desarrollo de este proyecto intentamos poner en práctica conocimientos de:
>- Programación orientada a objetos 
>- Patrones de modelo de diseño (Observers, Singleton)
>- Bases de datos (SQLite)
>- Desarrollo de interfaces gráficas (Tkinter)
>- Generacion de reportes (Reporlab)
>- Trabajo en equipo utilizando la herramienta GIT

---
#### Características que presenta el programa desarrollado

>- Administración de socios
>- Administración de libros
>- Registración de préstamos (devoluciones)
>- Registración de libros extraviados

**Generacion de Reportes:**
>- Cantidad de libros en cada estado
>- Sumatoria del precio de reposición de todos los libros extraviados
>- Nombre de todos los solicitantes de un libro en particular identificado por su título
>- Listado de préstamos de un socio identificado por su número de socio
>- Listado de préstamos demorados
